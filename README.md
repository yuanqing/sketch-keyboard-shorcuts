# sketch-keyboard-shortcuts

```sh
$ yarn global add sketch-keyboard-shortcuts
$ cat shortcuts.json
{
  "Arrange": {
    "Align": {
      "Bottom": "cmd ctrl b",
      "Left": "cmd ctrl l",
      "Right": "cmd ctrl r",
      "Top": "cmd ctrl t"
    }
  },
  "Layer": {
    "Flatten Selection to Bitmap": "cmd shift b",
  }
}
$ sks shortcuts.json
```
